# GenaRework

## Using

### Clone repository

```shell
git clone https://github.com/RashingPro/GenaRework.git
cd ./GenaRework
```

### Create `.env`

```env
TOKEN=bot_token
```

### Configure `config.json` Please use `config.schema.json`

### Build and run

```shell
npm install
npm run prod
```
